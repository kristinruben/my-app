* `react 15.6.1`
* `redux 3.7.2`
* `react-redux 5.0.5`
* `react-scripts 1.0.10`

## Installation
1. Clone the repo, `cd` into `my-app`, and run `npm install`.
2. `$ npm start`
3. Visit `localhost:3000` in your browser to view the app.
